# To-Do Manager

## Overview

A simple command-line To-Do Manager built using Python. Users can add, view, and delete tasks.

## Features

* Add Tasks
* View Tasks
* Delete Tasks
* File Storage
* Error Handling

## Technologies

* Python 3

## How to Run

```bash
python todo_manager.py
```

## Project Structure

* todo_manager.py
* tasks.txt
